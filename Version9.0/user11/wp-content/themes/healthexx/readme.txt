== Description ==

Healthexx is a modern WordPress Medical Theme created with the latest WordPress Coding Standards. It's created by using the latest Bootstrap Framework with HTML5 and CSS3 techniques and suitable for all popular mobile devices and tablets. To create a professional website, you can easily customize the theme sections like - Top header, Slider, Services, About us, Multiple Footer Widgets  and many more. If you are a non-techy person, you can easily modify the theme and design using live customizer in Dashboard > Appearance> Customize. You can change the color and the font of each and every text, link in the theme. This theme is translation ready.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
